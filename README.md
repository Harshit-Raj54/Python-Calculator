# Python Command-Line Calculator

A simple command-line based calculator developed using Python.  
The application performs essential arithmetic operations such as addition, subtraction, multiplication, division, and power operations through a user-friendly terminal interface.

---

## Features
- Supports addition (+), subtraction (-), multiplication (x), division (รท), and power (**)
- Handles invalid inputs using exception handling
- Prevents division by zero errors
- Interactive and easy-to-use command-line flow
